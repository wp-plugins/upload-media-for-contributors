== Description ==

This plugin adds the ability to upload media for the contributors of your site.

Just activate the plugin to make it work!


== Installation ==

1. Upload the complete `upload-media-for-contributors` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Should we set something to run the plugin? =

No, Just activate the plugin to make it work!
